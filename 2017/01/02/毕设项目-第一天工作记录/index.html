第一天安装Linux环境（Ubuntu），配置系统和语言环境比较轻松，但是配置Composer和Apache时各种问题。

安装Ubuntu

过程就不说了，很简单，主要记录下空间配置。因为电脑采用双硬盘（120G SSD+500G 机械）双系统（Win10+Linux），而引导需要放在主硬盘内，试了好多种方法，首先说明把Ubuntu引导放在副硬盘是绝对引导不了Linux系统的，其次，最好是采取grub2引导、先Linux后Windows。
最后记录下空间配置。因为我的方案是Win10+Ubuntu，两个系统都放在120G硬盘中，Win10占前80G，后40G（SSD大概大概只有110G，所以其实是35G左右）分给Ubuntu，分配方案如下：

  • /root：200M
  • /：20G
  • /home：15G
  • swap space：500M

另外，针对可能出现需要再分配空间，可以在Win10系统后空出2M作为BIOS启动空间。

配置Ubuntu主题

一直习惯Flatabulous主题，所以每次安装Ubuntu必安Flatabulous主题，原文地址：https://github.com/anmoljagetia/Flatabulous。

安装搜狗输入法

由于Ubuntu15.04版本后更改了默认的软件管理器，禁止了第三方兼容软件，所以安装起来变得困难很多，所以需要记住最基本的命令。

  1. 下载搜狗输入法Linux版：http://pinyin.sogou.com/linux/
  2. 命令行# dpkg -i sougou...<TAB>，如果出现缺少运行库的问题，那么直接跟着# apt-get install -f即可。

安装Vim

Linux系统下Vim是神器，我等渣渣只会最简单的操作，不过还是需要安装的，毕竟Vim是Linux系统下最强大的IDE，没有之一。
安装特别简单，一句# apt-get install vim即可，后面配置环境有用。

配置JDK环境

由于版权问题，Linux使用的并非是Orcale的JDK，而是OpenJDK，所以我们要自己下载安装JDK。

  1. 下载JDK
    上官网http://www.oracle.com/technetwork/java/javase/downloads/index.html下载对应的JDK（本文以`jdk-8u112-linux-x64.tar.gz`为例）。
  2. 配置JDK
    由于使用的特殊性，Linux需要修改两个地方的配置文件。一个是/etc/profile，这里是获得管理员权限后的环境配置文件；另一个是~/bashrc，这是用户的环境配置文件。两处加的内容都是一样，如下： 最后记得> source /etc/profile和> source ~/.bashrc一次。

安装IDEA、PHPStorm、Sublime一系列IDE

  1. IDEA和PHPStorm
    IDEA和PHPStrom是编辑器中的神器，自从接触后发现比eclipse好用很多，也比较习惯（Android Studio也是InteliJ公司出品的~）。上面配置完JDK后就可以安装这两款软件了（因为这是用Java写的）。
    下载好后tar -zxvf，然后sudo mv命令移动到/opt位置，进入/opt/<dir>/bin/后执行./xxx.sh即可打开，IDEA和PHPStorm都会自动配置图标和软链接，只需要输入两遍管理员密码即可。记得运行时要用用户权限，即>./idea.sh，不需要#./idea.sh。
  2. Sublime Text 3
    这个默认提供deb文件，下载好后用dpkg命令即可完成安装。至于Linux下无法输入中文的BUG，后面再讲。

安装常用软件

有道云音乐、有道词典

都是提供deb文件，dpkg命令安装即可。

配置APACHE

配置APACHE环境需要严格遵循命令行顺序！

下载Apache

下载地址：http://httpd.apache.org/download.cgi#apache24
也可以用命令

安装Apache

安装Apache前需要安装一些依赖库，最关键的是APR、APR-Util、PCRE三个。

  1. 安装APR
  2. 安装APR-Util
  3. 安装PCRE 如果上面步骤有错误，需要再安装g++环境。
  4. 安装Apache 安装好后修改Apache的端口： 如果用第三方集成包，可能是/usr/local/apache2/conf/路径，其实都是一样的。然后?Listen找到端口Listen 80，改成`Listen 8081（因为80端口被本地占用着）。最后验证Apache安装成功： 如果localhost:8081网页显示It works!证明安装成功。
