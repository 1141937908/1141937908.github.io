饥荒是真的好玩，从刚开始接触游戏的创意性到如今Mod的多样性，玩了这么久还是感觉最好玩的游戏。配完自己电脑上的环境后，闲着没事想起之前搭了很久没搭成功的服务器，于是又开始动手开始做这个了。

服务器配置

我用的是阿里云学生服务器，也就是9.9一个月的那个，所以配置是最低的，1G一内核+40G硬盘，不过搭小型饥荒服务器还是够的（大概够4-6人）。

操作系统: Ubuntu 14.04 TLS x64
CPU: 1核
内存： 1024 MB

Set cluster_token.txt,server.ini,worldgenoverride.lua

这三个是必需的配置，很简单，按照步骤来做就特别简单。

  1. 进入游戏
  2. 进入用户信息，添加一个token，用于服务器验证

  3. 把得到的token放进cluster_token.txt
  4. 添加worldgenoverride.lua到地上世界和地下洞穴两个服务中

    这里提供这两个文件的下载：Master/worldgenoverride.lua、Caves/worldgenoverride.lua。
    当然，这个还有更简单的方法，这个可以在本地创建一个世界，然后在自己的电脑本地（大概是我的文档\Klei\DoNotStarveTogether里）能找到，复制到服务器上也是一样的，而且更简单。（这个是参照百度贴吧）
