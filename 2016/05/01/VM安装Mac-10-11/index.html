此教程是源于百度的一个小汇总，可以作为安装Mac其他版本的参考样例。

安装配置参数

VMware版本：VMware Workstation Pro 12 （破解版）
Mac版本：OS X El Capitan 10.11
CPU核数：4
内存大小：4GB

必需文件

VMware Mac补丁：unlocker208.zip （提取密码：zvau）
Mac 10.11懒人版cdr：Install OS X El Capitan GM.cdr （提取密码：ps1h）

如果分享链接失败，则可以通过“2016/05/01”文件路径找到。

安装mac虚拟机

安装VMware

vmware（以下简称“VM”）这里我就不用说的了，直接到官网下载即可。注册机的话百度一下就行。

打Mac补丁

VM一开始不能识别Mac，或者说没有Apple Mac OS X系统这一选项，所以我们要打一个mac补丁。
首先直接将unlocker 208（以下简称“208”）解压到VM安装目录中，

这里说一下，本人试过在VM安装目录外直接安装，但是没有成功，所以建议将整个208放到VM安装目录下即可，还有一点，确保VM安装目录路径中没有任何中文！

接下来就是打补丁了。这里我借用别人的图说明一下在windows平台下安装过程：

后记

2016.5.1 关于Mac虚拟机无法连接物理网络的问题

Mac是安装完了，不过等我登陆进去的时候，发现无网络。百度搜索了下，发现需要架专用网络来连接。

网络适配器设置

在网络连接中选择物理网络适配器，右键设置——

然后选择共享选项标签，将两个选项都勾上，并且选择VMware Network Adapter VMnet1。（VMnet8是NAT模式，VMnet1是主机专用模式）

VM网络适配器设置

如图设置即可，然后重启虚拟机，查看Mac网络设置就能看到

安装VMare tools for mac

先下载一个VMware tools for mac，然后在虚拟机中修改CD/DVD设置，选中VMware tools的ISO文件，修改高级，将其设置为SCSI 0:12。

然后重启Mac虚拟机，安装即可。（说是少于一分钟，但是呵呵(￣▽￣)”）


文件共享

关闭Mac虚拟机，在虚拟机设置的【选项】标签中，启用共享文件夹。注意文件夹路径中不允许有中文！

全屏显示

这一点我还没成功，不过只能是每次全屏的时候大概开始有5~6秒有全屏，然后又变了……
