前言

Django框架入门

* 这部分对应书中第18章内容

当今的开发都是通过框架开发，因为讲究效率，而Python也不例外。Django作为Python首推Web框架，很轻松地能让你写出一个应用。下面就是根据书中所说，编写出了一个XX。

配置框架

对于不同应用开发需要不同版本的配置，Python提供了一种虚拟环境，可以很方便地让框架运行在所适配的环境中。首先我们就需要让框架跑在这种虚拟环境中，因此我们先安装virtualenv

安装virtualenv

这里需要注意几点：

  • 由于Python版本问题，所以安装时选择你想用的pip版本。
  • 另外，对于Linux系统，如果代码不管用可以使用系统内置的包管理器来安装，例如Ubuntu系统中可使用sudo apt install python-virtualenv安装。

安装完成后，就需要在项目目录中搭建虚拟环境配置，这样每次运行程序时候直接运行对应的虚拟配置信息，十分方便。激活虚拟环境也很简单，首先在目录中创建虚拟环境：
