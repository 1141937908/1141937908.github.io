前言

在《Python编程从入门到实践》一书中，对于数据可视化，采用了Python模块中的matplotlib库。

matplotlib作为一个利用Python 2D绘图库制作的工具，主要是用于绘制各类数据图形，而最常用的就是pyplot模块，因为pyplot模块提供了类似于MATLAB的方法接口，可以轻松的绘制图形，而且容易上手，我就是从书中一步步学习而来。但是由于思路不同，书中主要侧重模拟数据可视化的编码过程，对于matplotlib库的详细讲解相对弱化，而我现在所要做的就是总结书中出现的绝大多数函数和参数，对matplotlib常见函数能有个系统了解，并且以后能通过这篇文章直接找到所需要的内容，而不用在书中再去翻找相关代码片段了。

matplotlib.pyplot

matplotlib.pyplot是一个类似于MATLAB的绘图接口，提供各类基于状态的函数，主要用于交互式图形和程序化图形的生成。其最基本的操作首先就是引入接口：

如果想要查看所有API，可以访问：https://matplotlib.org/api/_as_gen/matplotlib.pyplot.html ，这里只总结下书上出现过的

Functions Description
scatter(x, y[, s, c, maker, cmap, norm, ...]) 绘制y关于x的离散图，可以标记不同大小和颜色
plot(*args, **kwargs) 绘制y关于x的线条图
savefig(*args, **kwargs) 保存当前图形

散点图/离散图 pyplot.scatter()

scatter()主要是绘制y关于x的散点图，通过各种参数进行调节，其主要需要提供绘制点的坐标。

Parameter Description
x, y 数据的位置（坐标）

当然，scatter()还有好多可选参数，但是这些参数对于pyplot接口来说都是通用的，所以我会在后面详细补充各类常见参数。

各类参数

对于matplotlib，由于是仿照MATLAB中对应函数设计的，所以参数基本也相同，可选参数在同一接口里都可以各种DIY，非常方便。而在文档中我们能看到的fmt可选参数，就代表着基本格式（如颜色、线条、标记等）的简写方式，行属性和fmt可以混用，产生的效果是相同的。关于2D行属性参数，具体可见matplotlib.lines.Line2D 。

这里我把书中所涉及到的散点图和线条图的参数总结下，两者参数区别不大。

Parameters Belong Description
c / color plot / scatter 即color，颜色（这块内容需要单独拎出来讲一下）
s plot / scatter 即shape，形状
marker plot / scatter 标记，默认为“o”，用来表示点或线的绘制形式（这块内容还需要单独拎出来讲下）
cmap plot / scatter 颜色映射（colormap），即一种颜色到另一种颜色的渐变
linewidth / lw plot 属于plot()，表示线条粗细
alpha plot / scatter 透明度，调节范围介于0.0~0.1之间
edgecolor scatter 数据点轮廓，默认为黑色

如果想查看详细参数介绍，可以查看官网对于两种图形参数的介绍：

  • 离散图 pyplot.scatter()
  • 线条图 pyplot.plot()

axes()

axes()函数还有各种子函数，实现关于坐标的各类功能，比如

Function Description
.get_xaxis() 获取x横坐标
.get_yaxis() 获取y纵坐标

xlabel()、ylabel()、title()

  • xlabel()：x横坐标名称
  • ylabel()：y纵坐标名称
  • title()：图表标题
  • show()：展示图片
